## Installation

### Dockerfile
 
 $ docker build --file Dockerfile --tag icip18_stats:0.1 .

### Start container

 $ docker run -it icip18_stats:0.1
 
### Run

 $ cd /root/src
 
 $ python gen_hist.py

 $ python gen_wordcloud.py
